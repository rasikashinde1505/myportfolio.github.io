Problem Statement:

 Website and Application design for the ‘TLS Bootloader’. ‘TLS Bootloader’ is basically developed to secure the connection between embedded device (in this case washing machine) and a user both will be connected to internet. The washing machine has a feature of software version update (like a smart phone updates).

• It has many additional features as fast washing, slow washing, different timings, kids cloth washing, different temperature settings and many more.
 • It also has version update feature. (must have)
 • Machine should have two options to download the update with TLS security and without TLS securities. (very must have)
 • To set alarm if machine stops to work by noticing the timer of the machine. (must have)
 • To set the alarm as the machine is done with the task.

From a user’s perspective:

I’m a working lady tries to keep my house clean everyday every time. But I always struggle with the home appliances like washing machines and fridges due to its very small hardware or software problems for which I always need to contact service helper to repair which cost more and takes lot of time. It would have been better if we can solve this problem handy.

From a user research perspective:

Busy working professionals need an easy, time-efficient way to make their house clean or to keep their stuff clean for which they get only weekend. If in between the washing machine or the electronic device sucks then they need to wait for it for long and can make the household work on the upcoming weekend. Because they often work for long hour’s and don’t have time.
